# 文学PDF祭（2025）

文学PDF祭は、2025年に発表された文学作品のPDFを集め、
ネット上で共有・保存するための企画です。

短歌・俳句・詩・小説・エッセイなど、ジャンルは問いません。
同人誌やネプリとして発表されたPDFも対象とします。

この企画は「文学コモンズ（Literary Commons）」の
初回の取り組みとして実施されました。

## 目的

- 紙や即売会に依存しないかたちで文学作品に触れられる場をつくる
- 2025年に発表された作品を、あとから辿れる形で残す
- 個人制作の成果を、一時的ではなく蓄積として共有する

## 収録対象

- 2025年に発表された文学作品のPDF
- ネプリ用PDF、同人誌PDF、個人制作PDF など
- 商業・非商業は問いません

## ディレクトリ構成

- `works/`  
  この企画で収録されたPDFの実体を保存します。

## 命名ルール

- ファイル名は投稿順の連番とします。  
  例：`001.pdf`, `002.pdf`, `003.pdf`

- タイトルやジャンルなどの情報は、
  README や上位ディレクトリから参照されます。
  ファイル名自体に意味を持たせる必要はありません。

## 補足

このディレクトリはアーカイブを目的としています。
作品の実体はここに一箇所だけ保存し、
ジャンル別の整理や紹介は別途インデックスで行います。
