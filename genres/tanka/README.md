# 短歌（Tanka）

このディレクトリは、日本語短歌作品の総合的な入口（インデックス）です。

短歌作品の実体（PDF・テキストなど）は、企画やアーカイブごとに
それぞれの場所に保存されています。
ここには作品そのものは置かず、参照・案内のみを行います。

## 構成

- 既存の短歌アーカイブ  
  過去半年分以上の短歌作品を収録している、既存のアーカイブです。

- 文学コモンズ内の短歌  
  「文学コモンズ」企画の中で発表された短歌作品の一覧です。
